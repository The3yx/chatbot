Step welcome
    Speak $name 您好，请问有什么可以帮您？
    Listen 20
    Branch 投诉 complainProc
    Branch 账单 billProc
    Branch 余额 balanceProc
    Silence silenceProc
    Default defaultProc
Step complainProc
    Speak 您的意见时我们改进的动力，请问您还有什么补充？
    Listen 20
    Silence silenceProc
    Default thanks
Step thanks
    Speak 感谢您的来电，再见。
    Exit
Step balanceProc
    Speak 您在本店余额为 $balance 元
    Exit
Step billProc
    Speak 您本月的账单为 $amount 元，感谢您的来电，再见！
    Exit
Step silenceProc
    Speak 听不清，请您大声一点
    Listen 20
    Branch 投诉 complainProc
    Branch 账单 billProc
    Silence silenceProc
    Default defaultProc
Step defaultProc
    Speak 不好意思，我不能理解您的意思，不好意思
    Exit
